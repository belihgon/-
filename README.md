# Евгений — Муж на час

Лендинг для мастера бытовых услуг в Архангельске и области.

## О проекте

Одностраничный сайт (лендинг) для услуги «Муж на час». Содержит:
- Информацию об услугах (сантехника, электрика, сборка мебели, дачные работы, клининг, мелкий ремонт)
- Цены на основные работы
- Галерею работ с лайтбоксом
- Форму обратной связи с валидацией
- Адаптивный дизайн под все устройства
- Микроразметку Schema.org и Open Graph для соцсетей

## Технологии

- HTML5
- CSS3 (адаптивная верстка, CSS-переменные)
- JavaScript (чистый, без фреймворков)
- Font Awesome 6 (иконки)
- Google Fonts (Inter, Montserrat)
